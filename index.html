Trustworthy Open Source

Open Source Software (OSS) constitutes the foundation of the modern digital economy. However, this ubiquitous integration has created a compelling vector for malicious actors. Modern systems are increasingly targeted through their OSS components, where the complex supply chain offers a strategic gateway for injecting malicious code.

BAGAD addresses this vulnerability by proposing a holistic framework to secure the OSS lifecycle—from dependency analysis to web and mobile privacy enforcement.

"We position our work as a form of Digital Public Health, providing vaccines (automated repair) and diagnostics (auditing) for the digital commons."

Our Research Program

Structured around three complementary axes

Supply Chain Security

Objective: Guaranteeing the provenance, integrity, and compliance of open-source components.

  • Secure SBOMs & Dependency Graphs
  • Detection of Shadow Dependencies
  • Sovereignty & Auditability

Automated Repair

Objective: Reduce "Time-to-Fix" by automating vulnerability discovery @scale and patch generation using Hybrid AI.

  • Large scale discovery using HyperAST
  • Large scale discovery using fast classifier
  • Self-Healing Bot in DevSecOps Pipeline

Privacy Enforcement

Objective: Auditing source code and enforcing privacy properties in Web and Mobile applications.

  • GDPR Compliance Checking
  • Data Leak Detection
  • Anti-Fingerprinting
